Автоматизированное тестирование сайта Ростелеком (https://start.rt.ru/?tab=main)

Описание проекта

Этот проект содержит автотесты для тестирования функционала сайта [Ростелеком](https://start.rt.ru/?tab=main). Тесты написаны с использованием **Selenium** и **PyTest** и покрывают основные сценарии взаимодействия с сайтом, такие как регистрация, вход, восстановление пароля, добавление товаров в корзину, оформление заказа и т.д.

Структура тестов

Файл `test_rostelecom.py` содержит 18 авто-тестов, каждый из которых проверяет отдельный функционал сайта:
- Проверка успешной регистрации
- Проверка входа с правильными и неправильными данными
- Восстановление пароля
- Редактирование профиля
- Проверка фильтров и добавление товаров в корзину
- Оформление заказа
- И другие тесты

Установка и настройка

Шаг 1: Установка необходимых библиотек

Убедитесь, что установлены **Python 3**, **Selenium**, и **PyTest**. Если они не установлены, выполните команду:
```bash
pip install selenium pytest
Шаг 2: Установка ChromeDriver
Узнайте версию вашего браузера Chrome (Откройте Chrome → Три точки в правом верхнем углу → Справка → О браузере Google Chrome).
Скачайте соответствующую версию ChromeDriver с официального сайта ChromeDriver.
Поместите chromedriver в папку проекта или в директорию, доступную через PATH.
Шаг 3: Запуск тестов
Перейдите в папку проекта с тестами:

bash
Копировать код
cd путь/к/папке/проекта
Запустите тесты командой:

bash
Копировать код
pytest test_rostelecom.py
Pytest выполнит все тесты и покажет результаты в терминале. Если тесты прошли успешно, вы увидите сообщение об успешном завершении тестов.

Примечания
Убедитесь, что сайт https://start.rt.ru/?tab=main доступен и работает, чтобы тесты выполнялись корректно.
Некоторые тесты могут требовать задержки для полной загрузки страницы. При необходимости добавьте time.sleep() для ожидания элементов.
